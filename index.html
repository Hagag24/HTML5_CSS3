HTML

HTML (لغة ترميز النص التشعبي) تُستخدم لتحديد هيكل صفحات الويب. فيما يلي بعض المفاهيم الأساسية:

العناصر الأساسية:

  • العناوين: تُستخدم لتمييز النصوص حسب الأهمية. مثال:

    هذا عنوان رئيسي H1

    هذا عنوان فرعي H2

  • الفقرات: تُستخدم لكتابة نصوص. مثال:

    هذه فقرة توضح كيفية استخدام عنصر <p> في HTML.

  • الروابط: تُستخدم للانتقال بين الصفحات. مثال: اضغط هنا لزيارة Example
  • الصور: تُستخدم لإضافة محتوى مرئي. مثال:
  • الجداول: تُستخدم لعرض البيانات المنظمة. مثال:
    اسم المنتج السعر
    منتج 1 100 جنيه
    منتج 2 150 جنيه
  • القوائم:

    قائمة مرتبة:

    1. العنصر الأول
    2. العنصر الثاني
    3. العنصر الثالث

    قائمة غير مرتبة:

    • عنصر 1
    • عنصر 2
    • عنصر 3

CSS

CSS (أوراق الأنماط المتتالية) تُستخدم لتنسيق صفحات HTML وتحسين مظهرها. تتضمن الميزات:

الخصائص الأساسية:

  • الألوان: تغيير لون النصوص والخلفيات باستخدام خاصية color و background-color.
  • الحدود: تخصيص حدود العناصر باستخدام خاصية border.
  • الحشوات والهوامش: التحكم في المسافات داخل وخارج العناصر باستخدام padding و margin.
  • التخطيطات: تنظيم العناصر باستخدام flexbox و grid.
