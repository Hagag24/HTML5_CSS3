HTML: اللغة الأساسية لتطوير صفحات الويب

HTML (لغة ترميز النص التشعبي) هي أساس أي صفحة ويب. تُستخدم لتحديد هيكل ومحتوى الصفحة. يتم تنظيم الصفحة باستخدام عناصر مثل:

  • <html>: العنصر الجذري. خصائصه:
    • lang: لتحديد اللغة الافتراضية للصفحة.
    • dir: لتحديد اتجاه النص (ltr أو rtl).
  • <head>: يحتوي على معلومات الميتا والروابط. خصائصه:
    • لا يحتوي على خصائص مباشرة، لكنه يضم عناصر مثل <meta> و <title> و <link>.
  • <body>: يحتوي على المحتوى المرئي. خصائصه:
    • background: لتحديد صورة أو لون الخلفية.
    • text: لتحديد لون النص الافتراضي.

عناصر HTML الأساسية

تتضمن العناصر الأساسية:

  • النصوص: باستخدام <p> و <h1> إلى <h6>. خصائصها:
    • align: لتحديد محاذاة النص (left, center, right).
  • الصور: باستخدام <img>. خصائصها:
    • src: مسار الصورة.
    • alt: نص بديل يظهر إذا تعذر تحميل الصورة.
    • width و height: لتحديد أبعاد الصورة.
  • الجداول: باستخدام <table>. خصائصها:
    • border: لتحديد عرض الحواف.
  • القوائم: مرتبة وغير مرتبة باستخدام <ul> و <ol>. خصائصها:
    • type: لتحديد نوع القائمة (دائري، رقمي، روماني).
    • start: لتحديد بداية الترقيم.

النماذج (Forms)

النماذج تُستخدم لجمع بيانات المستخدم:

CSS: لغة التنسيق

CSS تُستخدم لتحسين مظهر صفحات HTML. يمكن تطبيق CSS بطرق مختلفة:

  • Inline: داخل وسم HTML باستخدام السمة style.
  • Internal: داخل <style> في ملف HTML.
  • External: باستخدام ملف خارجي.

خواص CSS الأساسية

  • الألوان والخلفيات: باستخدام background و color.
  • الهوامش والحواف: باستخدام margin و padding.
  • الخطوط: باستخدام font-family و font-size.

الصندوق النموذجي (Box Model)

كل عنصر HTML يُعتبر صندوقًا يتكون من:

  • المحتوى (Content).
  • الحشوة (Padding).
  • الحافة (Border).
  • الهامش (Margin).

تكامل HTML و CSS

لربط CSS بصفحة HTML، يمكنك استخدام:

  • ربط خارجي: باستخدام <link> داخل <head>.
  • ربط داخلي: باستخدام <style> داخل <head>.
  • ربط داخلي: باستخدام السمة style داخل العنصر.

مثال:

<link rel="stylesheet" href="styles.css">
